 Used Cars Data Analysis 🚗

📌 Project Overview

This project analyzes used car listings to identify best-value vehicles based on price, mileage, and brand.

## 🛠 Tools Used

* Python (Pandas, BeautifulSoup, Selenium)
* SQL
* Power BI

## 📊 Key Steps

* Collected 1,650+ listings using web scraping
* Cleaned and processed data using Pandas
* Performed analysis to identify trends
* Built an interactive Power BI dashboard

## 🔍 Key Insights

* Identified brands offering best value for money
* Price variation depends on year and fuel type
* Reduced manual comparison time from 50+ hours to 30 minutes

* Dataset
* Python code
* Power BI dashboard
